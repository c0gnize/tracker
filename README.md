# Запуск

```sh
npm install
npm run start
```

# Задать специфичный адрес mongodb

Создать файл `.env`
```sh
cp .env.sample .env
```
и отредактировать переменную `MONGO_URL`

Проблемы:
- Был занят порт 5000 на mac
- Оказалось что CORS не работает, решил только после прочтения https://github.com/koajs/cors/issues/70
- Сначала файлы генерировал с помощью tsc. Но он упорно вставляет `export {}` в конец файла, а такие файлы выполняются только с `type="module"`, которые выполняются асинхронно после загрузки, т.е. после кода `tracker.track('pageview')`, что вызывает ошибку.Больше всего времени потратил на эту ошибку. В конце концов выбрал esbuild для сборки бандла.
